Landing Page — Curso Online de Investimentos

Landing page responsiva desenvolvida com foco em conversão, captação de leads e experiência do usuário.
Projeto ideal para cursos online, produtos digitais e validação de ideias.

Features

Página única (landing page)

   Layout responsivo (mobile-first)
   Copy orientada à conversão
   CTAs estratégicos
   Formulário com validação
   Envio de dados via Fetch API
   Feedback visual de sucesso/erro
   Menu mobile e scroll suave
   Tech Stack
   HTML5
   CSS3
   JavaScript (ES6+)

Run Locally
npm install
npm start


ou

python -m http.server 8000

Notes

Projeto front-end com escopo objetivo, sem backend próprio.
Indicado para portfólio, MVPs e landing pages comerciais.
