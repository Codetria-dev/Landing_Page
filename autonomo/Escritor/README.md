Landing Page Responsiva com Formulário de Contato

Descrição:

Este projeto consiste em uma landing page moderna, responsiva e funcional, desenvolvida com foco em conversão, ideal para apresentação de serviços, produtos ou profissionais autônomos.

O projeto demonstra a criação de uma página única com formulário de contato validado e envio de dados via Fetch API.

Funcionalidades
Página única (landing page)
Layout responsivo (mobile e desktop)
Seções de apresentação e chamada para ação (CTA)
Formulário de contato com validação básica
Envio de dados via Fetch API
Feedback visual de sucesso ou erro no envio

Tecnologias Utilizadas:

HTML5
CSS3
JavaScript (ES6+)

O que este projeto NÃO inclui:

Back-end próprio
Banco de dados
Autenticação
Painel administrativo
Animações complexas
SEO avançado
Deploy em produção

Clone o repositório
